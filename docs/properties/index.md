# Properties

The property pages support the current document-first Brando Schema model.

The most important shared property in the current site is `brando:jsonLdSidecar`, which links a Markdown and YAML-authored document to its machine-readable JSON sidecar.

## Property groups

| Group | Purpose |
| --- | --- |
| [Brand & context](brand-and-context.md) | Background brand and operating context fields. |
| [Expression tokens](expression.md) | Expression and identity-related fields retained from the broader vocabulary. |
| [Governance & policy](governance.md) | Governance, review, and control fields. |
| [Product & category](product-and-category.md) | Product and category alignment fields. |
| [Runtime & campaigns](runtime-and-campaign.md) | Campaign and execution-oriented fields. |
| [Shared properties](cross-type.md) | Fields that can be reused across multiple entity types, including `brando:jsonLdSidecar`. |
