# Partners

Brando Schema is intended to be open, implementation-friendly documentation for teams that want to structure brand and governance documents in a consistent way.

## Partner types

We distinguish between two broad partner groups:

- implementation partners who help organisations model, author, and govern document entities
- technology partners who integrate the Markdown / YAML / JSON triad into platforms, pipelines, or operational systems

## What alignment means

Partners should be able to work with the current Brando Schema authoring model:

- document-first entity types
- consistent Markdown body, YAML frontmatter, and JSON sidecar handling
- linked property usage, including `brando:jsonLdSidecar`
- governance-aware content review and publishing workflows

## Trademark and compatibility

Partners may describe their services as compatible with Brando Schema, but should avoid implying ownership, endorsement, or exclusive status unless that has been agreed explicitly.
