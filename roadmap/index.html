    Brando Schema Roadmap and Working Groups

    Overview

    This page outlines the development roadmap for the Brando Schema project and defines active working groups focused on specific aspects of the initiative. Contributions are welcome across all areas, whether through code, documentation, schema proposals, research, community engagement, or governance.

    Roadmap

    Phase 1 – Foundation (Complete)

    • Define core schema classes: Verbal Identity, Visual Identity, Positioning, Governance, Assets
    • Launch MkDocs-based documentation site
    • Implement bdv-terms and bdv-classes pages
    • Draft 1.0 Brand Definition Vocabulary (BDV)
    • Register brandoschema.com and deploy site via GitHub Pages

    Phase 2 – Expansion (🔄 In Progress)

    • Expand schema to cover Audio Identity and Sonic Branding
    • Publish JSON-LD context file for BDV (v1.1+)
    • Implement structured FAQ and Roadmap pages
    • Seed GitHub Discussions and enable community Q\&A
    • Add support for multilingual branding schemas
    • Launch Brando Metadata Chrome Extension (MVP)

    Phase 3 – Tooling and Compliance (🔜 Planned)

    • Add SHACL and JSON Schema validation tools
    • Launch AI Visibility Auditing Toolkit (CLI + Cloud Agent)
    • Define Brando Brand Integrity Signals for LLM safety
    • Integrate Brando schema into LLM prompt scaffolding tools

    Phase 4 – Governance and Standardisation (🔜 Planned)

    • Formalise governance model and community charter
    • Submit Brando Schema to W3C Community Group
    • Partner with AI labs and regulators for AI-safe branding
    • Launch the Brando Brand Assurance Badge

    Working Groups

    1. Schema Design Group

    Focus: Designing and evolving the Brando Schema, validating terms, aligning with schema.org

    • Maintainers: Jonathan Bowker (Lead)
    • Tools: SHACL, JSON Schema, MkDocs
    • Meetings: Monthly schema review

    2. Tooling & Integrations Group

    Focus: Chrome extensions, AI auditing agents, plugin integrations, validation scripts

    • Open to contributors with Python/JS/LLM experience
    • Projects: BrandoBot, Metadata Validator, Prompt Assistant

    3. Community & Documentation Group

    Focus: FAQ development, tutorials, example brands, onboarding docs, use case libraries

    • Maintainers: TBD
    • Initial Tasks: Translate docs, create contributor guide, seed GitHub Discussions

    4. Compliance & Governance Group

    Focus: Alignment with AI regulations, brand safety, ethics, formal governance structures

    • Goals: Define assurance criteria, build trust models, engage with AI safety communities
    • Stakeholders: Legal, academic, and civic AI advisors

    5. Outreach & Adoption Group

    Focus: Partner outreach, brand engagement, ecosystem growth

    • Tasks: Create starter templates, conduct workshops, collect feedback from designers and strategists
    • Metrics: Number of brands structured, Chrome extension installs, schema visibility audits

    How to Join

    • Comment on a GitHub Discussion post for the relevant group
    • Submit a PR or issue with your interest or ideas
    • Follow @BrandoSchema on social platforms (coming soon)
    • Add yourself to the CONTRIBUTORS.md file via PR
